## 2.4turtle程序语法元素分析

1.库引用
    
    使用import保留字完成，采用<a>.<b>()编码风格
        import <库名>
        <库名>.<函数名>(<函数参数>)
    
    使用from和import保留字共同完成
        from <库名>import <函数名>
        from<库名>import *
        <库名>(<函数参数>)
        
    上述两种方法比较：第一种方法不易出现函数重名的方式；如果程序很短，可以使用第二种方法
    
    使用import和as保留字共同完成
        import<库名>as<库别名>
        <库别名>.<函数名>(<函数参数>)
    这个方法冗余代码量最少，又避免了函数容易重名的问题
